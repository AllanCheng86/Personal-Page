Weather Application

Developed a weather application that allows the user to enter a city name. The server then translates the city name into geographical coordinates using openweathermap's geocoding API and fetches weather data from openweathermap's weather API. The JSON data gets parsed and is populated as HTML using EJS. The back end is created using Node.JS and Express.JS. HTML is styled using only CSS.
